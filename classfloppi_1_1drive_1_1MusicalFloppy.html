Detailed Description

Class for controlling one musical floppy drive.

This class controls one floppy drive in its own thread, so several drives can be used in parallel.

  interval,
  count,
  rest 
)

Shake the head with a certain interval.

Moves the head back and forth some times, pausing in between. You can make sounds at certain frequencies by setting the interval to 1/freq. Adjust the duration of the vibration by setting the count to freq*duration.

Parameters
selfthe object pointer
intervalthe pause between movements, in seconds
counthow often to move the head
resthow long to wait to compensate rounding problems
