135  ## Shake the head with a certain interval
136  #
137  # Moves the head back and forth some times, pausing in between.
138  # You can make sounds at certain frequencies by setting the interval
139  # to 1/freq. Adjust the duration of the vibration by setting
140  # the count to freq*duration.
141  #
142  # @param self the object pointer
143  # @param interval the pause between movements, in seconds
